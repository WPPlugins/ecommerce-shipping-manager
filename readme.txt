== Description ==

Shipping Manager for WooCommerce allows merchants to easily upload CSV files that contain order ID's and tracking numbers. Merchants no longer
have to spend countless hours manually updating orders once the product has shipped.

= Features =
- Easy file upload
- Reads CSV files
- Optional customer emaial sent with tracking update
- Customize order note and email sent to customer
- Select file delimiter
- Debug log for troubleshooting
